This is a flask app for searching and viewing job 
listings and creating new job listings

Dependencies which are required to be installed in 
the environment in which this app is run are listed 
in the dependencies.txt file. 

project directory should be as follows:

<app_folder>
    static
        style.css
    templates
        base.html
        create.html
        home.html
        listing.html
        search.html
    .flaskenv
    app.py
    best_svm_model.pkl
    googleNews300_w2v.model
    googleNews300_w2v_model.vectors.npy
    jobs_clean.txt
    README.txt
    requirements.txt
